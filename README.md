# Ejercicios y prácticas técnicas

Repositorio que agrupa ejercicios, prácticas técnicas y trabajos académicos desarrollados durante la formación universitaria en Ingeniería en Computación. El contenido refleja procesos de aprendizaje, experimentación y análisis en distintas áreas de la computación.

## Contenido
- Implementaciones de algoritmos básicos y avanzados
- Estructuras de datos
- Simulaciones y pruebas de conceptos de programación
- Ejercicios prácticos en distintos lenguajes
- Prácticas de Human-Computer Interaction (HCI)
- Análisis conceptuales y visuales sobre tecnología e inteligencia artificial
- Prototipos, wireframes y documentación académica

## Áreas abordadas
- Programación y fundamentos computacionales
- Human-Computer Interaction (HCI)
- Diseño centrado en el usuario
- Inteligencia Artificial (enfoque conceptual y académico)
- Investigación, desarrollo e innovación (I+D+i / I+D+A)

## Contexto
Este repositorio tiene fines académicos y de referencia técnica. Los ejercicios y trabajos aquí contenidos no representan proyectos profesionales completos, sino prácticas orientadas a fortalecer la base conceptual, analítica y técnica en distintas áreas de la computación.

Algunos trabajos incluyen múltiples versiones, las cuales reflejan procesos iterativos de mejora, retroalimentación académica y evolución del contenido.

## Tecnologías y herramientas utilizadas
- C / C++
- Python
- Java (según aplique)
- Markdown
- Herramientas de prototipado (según la práctica)
- Otros lenguajes y herramientas según el ejercicio
