# Ejercicios y prácticas técnicas

Repositorio que agrupa ejercicios y prácticas técnicas desarrolladas durante la formación universitaria en Ingeniería en Computación.

## Contenido
- Implementaciones de algoritmos básicos y avanzados
- Estructuras de datos
- Simulaciones y pruebas de conceptos de programación
- Ejercicios prácticos en distintos lenguajes

## Contexto
Este repositorio tiene fines académicos y de referencia técnica. Los ejercicios aquí contenidos no representan proyectos profesionales completos, sino prácticas orientadas a fortalecer la base conceptual y técnica.

## Tecnologías utilizadas
- C / C++
- Python
- Java (según aplique)
- Otros lenguajes según la práctica
