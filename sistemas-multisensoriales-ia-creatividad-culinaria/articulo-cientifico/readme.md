# Artículo científico

Artículo académico titulado:

**“Sistemas Multisensoriales e IA Generativa para la Creatividad Culinaria: Integración Multimedia y una Rueda de Sabores en Realidad Aumentada”**

## Descripción
El artículo analiza la integración de sistemas multisensoriales, inteligencia artificial generativa y tecnologías de realidad extendida aplicadas al ámbito culinario, con énfasis en la creatividad, personalización y experiencia del usuario.

## Enfoque
- Revisión de literatura científica
- Diseño conceptual de sistemas interactivos
- Prototipado con tecnologías inmersivas
- Evaluación de experiencias de usuario

## Tipo
Trabajo académico / artículo científico (formato revista)
