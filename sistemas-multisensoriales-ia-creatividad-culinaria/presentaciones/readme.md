# Presentaciones académicas

Material utilizado para exposiciones académicas relacionadas con sistemas multisensoriales, inteligencia artificial generativa y creatividad culinaria.

## Contenido
- Diapositivas de presentación
- Resumen visual de metodología, hallazgos y conclusiones

## Propósito
Apoyar la divulgación académica del trabajo de investigación y facilitar la comunicación de conceptos complejos de forma visual y estructurada.
